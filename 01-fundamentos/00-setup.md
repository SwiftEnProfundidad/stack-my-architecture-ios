# Setup: Preparación del entorno

## Objetivo

Dejar una base local estable para empezar la Etapa 1 sin fricción de tooling.

## Checklist rápido

- Xcode actualizado (Swift 6.2 disponible).
- Simulador iOS operativo.
- Git configurado.
- Proyecto compila y ejecuta una app base en simulador.
- Target de tests creado y ejecutable (`Cmd + U`).

## Verificación mínima

1. Crear proyecto iOS en Xcode.
2. Ejecutar build (`Cmd + B`).
3. Ejecutar app (`Cmd + R`).
4. Ejecutar tests (`Cmd + U`).

Si este baseline no está en verde, corrígelo antes de pasar a estructura Feature-First.
